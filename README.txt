The objective of this project is to build a microservice system that provides the information for all available courses in one place/application and allows comparison between the courses and modules that they contain provided by the universities in the United Kingdom and Great Northern Ireland. The data required for this web application has been gathered by web scraping.

Technology stack: Eclipse IDE, Java programming language, MongoDB, Spring boot, Spring MVC, RESTful, JSoup library, Maven,CSS,HTML;
